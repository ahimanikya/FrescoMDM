Configuring Master Indexes

The topics listed here provide information about configuring a master index application. You can perform the tasks listed on this page after you create the master index framework. This document is designed to be used in conjunction with Understanding Master Index Configuration Options, which describes the configuration file structure and explains the configuration options and how they affect processing.
