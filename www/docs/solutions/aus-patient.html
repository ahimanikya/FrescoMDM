Description

The purpose of this solution is to demonstate how an Australian Patient model might be implemented in the Master Data Management (MDM) product suite. Specificaly the Master Index portion of MDM.

This solution uses a patient definition geared towards the Australian health care system. Since this is an AUSPatient, fields such as the First Name, Last Name, Date of Birth and the Tax File Number (TFN) are used for both block-search definitions and matches.

Data Model Overview.

This overview of the data model shows each Sub Object Node of the AUSPatient. It highlights the main sections of the Australian Patient model. Additionally, it shows the required fields of the model.


Data Model

This is the UML model used to create the Australian Patient. You will notice that the model shows inheritance since the AUSPatient and the United States of America Patient models are very similiar. The main difference being the use of the Tax File Number (TFN) in Australia and the Social Security Number (SSN) in the United States as a unique identifier. Additionaly, this model represents differences in the health care system reflected by the HeathFund Sub Object Node of the Austriallian Patient.

User Codes

When the user-code is selected in the MIDM GUI then the coresponding format, input_mask and value_mask are used. The input_mask shows the characters that can be typed.

For the input_mask and value_mask:
A = an alphanumeric character [A-Za-z0-9]
D = a decimal number [0-9]
L = an alphabetic character [A-Za-z]

For the value_mask:
^ = remove the corresponding character from the input string when storing in the data.


Example: The AUXIDDEF shows three values in the MIDM GUI, "CREDIT CARD ID", "Medical License ID" and "Household ID". If the user selects the Household ID they can type in four deciimal numbers, a space, four deciimal numbers, a space and then four deciimal numbers (example: 1111 2222 3333 4444). This will be converted to 1111222233334444 and stored in the database. 
insert into sbyn_user_code (code_list, code, descr, format, input_mask, value_mask)
 values ('AUXIDDEF', 'CCID', 'CREDIT CARD ID', '[0-9]{16}', 'DDDD DDDD DDDD DDDD', 'DDDD^DDDD^DDDD^DDDD');

insert into sbyn_user_code (code_list, code, descr, format, input_mask, value_mask)
 values ('AUXIDDEF', 'MEDID', 'Medical License ID', '[A-Za-z]{3}[0-9]{6}', 'LLL-DDDDDD', 'LLL^DDDDDD');

insert into sbyn_user_code (code_list, code, descr, format, input_mask, value_mask)
 values ('AUXIDDEF', 'HHID', 'Household ID', '[A-Za-z0-9]{3}[0-9]{4}', 'AAADDDD', 'AAADDDD');
