Data Model Overview

This  overview of the data model shows each Sub Object Node of the AUSPatient.  It highlights the main sections of the Austrlian Patient model.  Additionally, it shows the required fields of the model.

Data Model

This is the UML model used to create the Austrilian Patient.   You will notice that the model shows inheritance since the Austrilian Patient  and the USA Patient  models are very similiar. The main difference being the use of a Tax File Number (TFN) in Austrlia and the as an unique identifier Social Security Number (SSN) in the United States as an unique identifier
