Description

The purpose of this solution is to demonstate how a United States of America Citizen model might be implemented in the Master Data Management (MDM) product suite. Specificaly the solution focuses on the  Master Index portion of MDM.

Data Model

The following diagram displays the USA Citizen model. USA Citizen object contains attributes which are not repeated and have zero or one occurrence. Attributes in bold are required. Sub-objects such as Alias and Address are linked to the USA Citizen object. Sub-objects have zero to many occurrences. For each sub-objects to exists, required attributes in bold for that sub-object must be present.
