Installing Mural Builds

Last Updated: May 2008

This page provides the information and instructions you need to download the Mural applications and install them into a NetBeans environments. It includes the following topics:

  • What's Included?

  • Prerequisites

  • Downloading a Mural Build

  • Getting Started With Mural

What's Included?

The Mural products are packaged into an NBM file that you can install directly into NetBeans. The installation currently includes the following products:

  • Master Index Studio

  • Mural Match Engine

  • Mural Standardization Engine

  • Data Profiler

  • Data Cleanser

  • Initial Bulk Match and Load Tool

You can find a list of documentation for Master Index Studio and the Initial Bulk Match and Load Tool at https://open-dm-mi.dev.java.net/DocumentsByTitle.html. For the remaining products, a list of documentation is at https://open-dm-dq.dev.java.net/DocumentsByTitle.html.

Prerequisites

Before you install the Mural build, install the following products on your computer.

  • NetBeans 6.1 – Download the complete version from http://www.netbeans.org/community/articles/netbeans61.html.

  • Java SE Development Kit 5 – Download Java SE from http://java.sun.com/javase/downloads/index_jdk5.jsp.

  • Oracle or SQL Server database driver – Download the Oracle driver from http://www.oracle.com/technology/software/tech/java/sqlj_jdbc/index.html. Download the SQL Server driver from http://www.microsoft.com/downloads/details.aspx?FamilyId=6D483869-816A-44CB-9787-A866235EFC7C&displaylang=en.

Downloading a Mural Build

You can download three versions of the Mural build. The nightly build includes all the latest updates, but has not been tested and stability is not guaranteed. The stable build has been run through some testing and is more stable than the nightly builds. The early access build has passed thorough testing and is very stable.

  1. Navigate to https://mural.dev.java.net/Downloads.html. You can also access this page by clicking the Get Started! button on the Mural home page.

  2. Click the build you want to download.

  3. Save the file to your disk.

    Note: The name of this file should be mural.nbm, but might vary among different web browsers. If your browser downloads mural.zip or mural.nbm.jar, rename the file to mural.nbm.

  4. Start NetBeans, if it is not already running.

  5. From the NetBeans Tools menu, click Plugins.

  6. Click the Downloaded tab and then click Add Plugins.

  7. Navigate to the location where you saved mural.nbm, select the file, and then click Open.

  8. Click Install.

  9. On the Installer, click Next, select the license acceptance check box, and then click Install again.

  10. If a validation warning window appears, click Continue.

  11. When the installation is complete, click Finish and close the Plugins window.
