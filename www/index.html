The Open-DM-MI project provides the capability to create any domain specific master index through the matching, de-duplication, merging, and cleansing of data from various data sources.

Open Master Index is the true and reliable alternative to proprietary, closed-source MDM, Customer Data Integration (CDI), and product information management (PIM) products. Master Index leverages other open source projects and open standards, with extensive potential to jump-start and leverage auxiliary open-source efforts. Developers have the tools and runtime environment to create robust MDM applications



The Open-DM-MI project has these high-level features:  
 
 
Sub-component of Mural community that allows the creation of any domain specific master index.
Provides the capability to match, de-duplicate, merge, and cleanse data from various sources.
Creates a three-tier, J2EE indexing application based on a business object definition.
Provides flexibility through configuration.
Provides a pluggable framework for incorporating industry specific business logic.
