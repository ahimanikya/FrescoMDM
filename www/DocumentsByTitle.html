Documentation
June 2008

This page lists the user's guides and reference materials available for Open Master Index. Keep checking this site for the most recent information and documentation.


Before You Begin

  • Developing in Open Master Index
    Gives information and instructions you need in order to get started with Master Index Studio, including software requirements, repository access, and build instructions.

Note: Mural builds are available for you to download so you can begin working with Mural applications without having to build the applications yourself. Go to the Mural Download page to get started.


Solutions

  • Mural developers have created a set of master index models, which are predefined templates for a master index application based on specific industry requirements. These models provide samples of master index configuration for various business requirements. To learn more about the models, see Mural Industry Solutions.

User's Guides

  • Developing Master Indexes
    Gives instructions for creating a master index application and the master index database, and also includes instructions for configuring the database connection.
  • Configuring Master Indexes
    Gives instructions for configuring a master index application by either using the Configuration Editor or modifying the configuration files directly.
  • Loading the Initial Data Set for a Master Index Application
    Gives instructions on how to use the load tool to insert the initial data set into a master index database.
  • Working with the Master Index Data Manager
    Gives instructions on how to use the web-based interface to maintain data in a master index application.
  • Maintaining Master Indexes
    Gives information and instructions on how to maintain a master index application once it is in production.
  • Broadcasting Master Index Updates to External Systems
    Gives information and instructions on how to create a Topic that publishes any updates made to the master index database. Includes a BPEL process as an example.

Reference Material

  • Understanding Master Index Configuration Options
    Gives background information about the configuration options for a master index application and describes the XML files.
  • Understanding Master Index Processing
    (Coming soon!) Gives information about the default database structure, the dynamic Java API, and how data is processed.

  • Understanding the Mural Match Engine
