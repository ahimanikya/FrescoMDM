Manage

Used to primarily view existing relationships and history.

Used to maintain the relationships (add, delete and where appropriate move) though this is most likely a lot rarer. Indeed it is expected that the various 'relationships' will be populated by rules at run time.  The data stewart is most likely to use the maintanance features to view and rarely fix an erronous relationship.
