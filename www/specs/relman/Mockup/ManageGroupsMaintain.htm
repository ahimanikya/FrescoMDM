Manage->Groups->Maintain

Displays information about records and how they are grouped. Either a user can find records associated with a group or find the groups that a specific record is associated with.

Use Cases

  • Maintain groups of records.

    An example of a group is household.  Where the household group shows all the members of the same household.
  • View the groups associated with a specific record.
  • View the records within a group.
  • View details about a specific records within a group.
  • View the properties (fixed and extended attributes) associated with a specific record in a group.

Records in a Group


  • User wants to find records associated with a group.
  • User may want to add records to the selected group.
  • User may want to remove records from a selected group

Notes


  • If no records are associated with a group the table should have a single 'No Records Found' entry.
  • The View Associated Groups button should only be enabled if at least one different group is associated with the same record. Also the associated groups must be viewable based on the current access role.
