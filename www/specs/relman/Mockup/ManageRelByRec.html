Manage->Relationship->Maintain->By Record  

Use Cases

  • View and maintain relationships for a single record and all relationships to all associated records.
  • View details of selected relationship (source record, destination record and relationship attributes)

Notes on Relationship Model overview


  • The windows should be minized by default

Notes on Tree Usage


  • Change the Relationship Model when user selects a record deeper in the tree than the currently selected record 
  • Selecting a relationship type allows deletion of all the records in the relationship.
  • Selecting a domain disables Delete and Add.
  • Allow for multi-selected.  If more than one record is selected then the right hand side should be blank.  
  • Do not allow expantion of any record that is already expanded in the current  tree path.

    Example:  Below "Dr. George Karlin" is not expandable because the record is already expanded on the tree.

  • Eventually prototype support selection of several nodes in the same relationship for use in Re-assign
