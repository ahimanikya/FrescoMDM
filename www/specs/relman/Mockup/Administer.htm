Administer

Administer relationship, hierarchy, groups and/or categories.

This includes:

  • Defining the relationship types (domains, name, ...)
  • Define custom attributes associated with a record in a relationship or with a Group Instance
  • Defining a taxonomy for categories. IE what category is a sub-category and so on.
  • Define if a custom attribute is  Searchable.  Custom attributes legal Searchable values are   ' By Value', 'By Range' or 'No'

    If an attribute is Searchable by Range show a Drop-Down list by the field any dialog that allows a user to search for a Relationship with the selection of Search by: 'By Value', 'By Range'.  By Value shows the entry field by the attribute and By Range shows From: entry-field To: entry-field.

    Once a selection is made the Web GUI should remember the preference by Association and Custom Attribute for the Session.
