Administer

Administer relationship, hierarchy, groups and/or categories.

This includes:

  • Defining the relationship types (domains, name, ...)
  • Define custom attributes associated with a record in a relationship or with a Group Instance
  • Defining a taxonomy for categories. IE what category is a sub-category and so on.
  • Define if a custom attribute is  Searchable.

Handling of Searchable Attributes

The exact attributes that are shown on a search screen are defined in the Netbeans Configuration.  However, if no search screens definition exist then each searchable attribute is shown.   Besides each search screen is an option to "Search by Range".  If selecetd the attribute allows the user to specify a range of values to search on.
