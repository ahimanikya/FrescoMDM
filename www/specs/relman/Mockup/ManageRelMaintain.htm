Manage->Relationship->Maintain->ByRecord

View and maintain existing relationships depending on the user's current access control. This allows exploration of relationships by showing associated relationships.

As with all the other screens details about a record and relationship properties can be viewed by selecting a record.


There are two ways that a user can maintain (view  / update)  Relationship Type instances.

Maintain - By Relationship

Use Cases

  • User wants to see records that are associated with a Relationship Type between two specific domains.
  • User selects the Relationship type and optionally supplies search criteria for records in the selected relationship.
  • User selects a relationship instance to view record summararies and view / update Relationship properties.

View Maintain by Relationship




Maintain - by Record  

Use Cases

  • View and maintain relationships for a single record and all relationships to all associated records.
  • View details of selected relationship (source record, destination record and relationship attributes)

Screens


1. Select by Record


When first selected a Search Dialog Overlay is shown with a message to Select the record to find relationships to.
