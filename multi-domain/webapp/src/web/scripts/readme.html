The multi-domain web manager utilizes the DoJo to implement the UI. The DoJo is an open source DHTML toolkit written in JavaScript. The Dojo 1.2.1 is the latest stable release dated on November 2, 2008 and downloaded from


http://download.dojotoolkit.org/current-stable/


It is licensed under the Academic Free License version 2.1. The final version supported by the multi-domain web manager depends on the Dojo release schedule.


Currently the following Dojo widgets are checked in:

  • dojo

  • dijit

  • dojoX/grid


more widgets from dojoX will be checked if they are selected to be utilized.
